UV DLP Slicer Controller

A slicer and print controller for UV DLP based 3d printers

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Creation Workshop

This is a program used for controlling your 3d UV DLP printer. It combines both a slicer and controller. It can control a number of printer types that utilize RAMPS-like hardware.
